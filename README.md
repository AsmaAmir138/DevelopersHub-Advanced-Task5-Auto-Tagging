# Advanced Task 5: Auto Tagging Support Tickets Using LLM Logic

This repository contains my submission for **Advanced Task 5** of the AI/ML Engineering Internship at DevelopersHub Corporation.

##  Objective
To auto-classify customer support tickets into relevant tags using Simulated LLM text-classification logic comparing Zero-Shot vs Few-Shot learning structures.

##  Implementation Details
- **Zero-Shot Learning:** Direct token extraction and semantic mapping onto targeted classification tags.
- **Few-Shot Learning:** Added few-shot examples into the classifier logic to enhance performance on unstructured ticket complaints.
- **Output Structure:** The model ranks and outputs the top 3 most probable tags for any input text.
